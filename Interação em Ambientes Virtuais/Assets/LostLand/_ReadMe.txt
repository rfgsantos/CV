Hello!
Everything is ready to go, just do some actions.

1. Open demoScene from LostLand foldes.
2. Set 2 additional layers: Terrain, Foliage.
3. Assign foliage to "foliage" layers and terrain mesh to "terrain" layer.
4. Run "Component/LostLand/Assign Terrain LOD"
5. Run "Component/LostLand/Prepare to Lightmapping"
6. In quality settings set shadow distance to higher values, 300-400.
7. click run.

Please, read user's manual before using.
